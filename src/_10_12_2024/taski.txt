1. Создание пользовательского класса
Создать класс CustomObject, который содержит следующие поля:

int id – идентификатор объекта.
double value – дробное значение.
boolean isActive – статус активности.
char symbol – символ.
float weight – вес объекта.
byte age – возраст объекта.
short level – уровень объекта.

2. Конструктор и методы класса
Конструктор должен инициализировать все поля.
Переопределить метод toString() для вывода информации об объекте в удобном формате.

3. Метод для генерации массива объектов
Создать метод generateObjects, который:

Возвращает массив из 20 объектов класса CustomObject.

Инициализирует поля случайными значениями:
id – уникальное значение (от 1 до 20).
value – случайное дробное значение (от 0 до 100).
isActive – случайное логическое значение.
symbol – случайный символ (латинский алфавит, A–Z).
weight – случайное значение от 1.0 до 50.0.
age – случайное значение от 0 до 127.
level – случайное значение от 0 до 32.767
